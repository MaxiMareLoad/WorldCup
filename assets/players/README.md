# 球员高清图（本地覆盖）

把球员的高清图放在这个目录，App 的球员详情页 hero 会**优先**使用本地图，
未提供时自动回退到懂球帝头像 / Wikimedia 大图。

## 命名方式（二选一，都支持）

- 按懂球帝 **person_id**：`50134172.png`
- 按**中文名**（与 App 显示一致）：`哈弗茨.png`

支持扩展名：`.png`（推荐，透明背景效果最佳）、`.webp`、`.jpg`、`.jpeg`。

## 推荐用脚本自动处理

把原图丢进 `assets/players_raw/`，运行：

```bash
python tools/process_portraits.py
```

脚本会自动抠图（透明背景）+ 调成深红电影风，输出到本目录。
详见 `tools/process_portraits.py` 头部说明。

> 不放任何图也没关系：App 现在会自动从懂球帝拉取每位球员（含未参加本届
> 世界杯的）的真实头像与完整资料，所有球员都有图、有资料、不黑框。
